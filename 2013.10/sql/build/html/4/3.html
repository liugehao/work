字符变量使用单引号，不要使用双引号¶

字符变量使用单引号，不要使用双引号，【”2012-09-23 00:00:00”】 可改为 【‘2012-09-23 00:00:00’】

SELECT emp_no, first_name, last_name, hire_date FROM `employees` WHERE hire_date="1986-06-26"

用单引号写成：

SELECT emp_no, first_name, last_name, hire_date FROM `employees` WHERE hire_date='1986-06-26'
