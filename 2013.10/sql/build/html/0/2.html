不要从明细表查统计结果，定期统计插入到汇总表¶

一般做法¶

SELECT
  `cxxx_id`,
  COUNT(1)
FROM
  `xcbd`
WHERE cxxx_id = 23057901

使用汇总表¶

建立汇总表：

CREATE TABLE `xcbd_cxxx_count` (
`cxxx_id` INT PRIMARY KEY,
`count_num` INT
);

定时插入数据进汇总表之前先清空汇总表：

TRUNCATE TABLE xcbd_cxxx_count;

定时插入数据进汇总表：

INSERT INTO `xcbd_cxxx_count` (`cxxx_id`, `count_num`)
SELECT
  `cxxx_id`,
  COUNT(1)
FROM
  `xcbd`
GROUP BY `cxxx_id` ;

查询汇总表数据：

SELECT
  cxxx_id,
  count_num
FROM
  xcbd_cxxx_count
WHERE cxxx_id = 23057901
