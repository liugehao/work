明细统计时，只统计编码，不要关联名称等冗余字段¶

SELECT
  bd.clxx_id,
  cl.cp,
  COUNT(bd.clxx_id)
FROM
  `xcbd` AS bd
  INNER JOIN clxx AS cl
    ON bd.clxx_id = cl.id

或者

SELECT
  clxx_id,
  COUNT(1),
  (SELECT
    cp
  FROM
    clxx
  WHERE id = xcbd.`clxx_id`) AS cp
FROM
  `xcbd`

正确做法

SELECT
  clxx_id,
  COUNT(1)
FROM
  `xcbd`

取出结果后，根据实际情况再取cp。cp, id可以考虑运用缓存存储
