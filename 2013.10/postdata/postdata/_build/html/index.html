post数据测试¶

环境¶

rhel5.7 2核虚拟机，4G内存

安装pycurl¶

yum install python-pycurl.x86_64

安装PHP¶

wget http://us1.php.net/get/php-5.5.5.tar.gz/from/cn2.php.net/mirror
tar zxf php-5.5.5.tar.gz
yum install libxml2-devel.x86_64
cd php-5.5.5
./configure
make -j 4
make install

安装php的curl扩展¶

cd php-5.5.5/ext/curl
phpize
./configure
make -j 4
make install
